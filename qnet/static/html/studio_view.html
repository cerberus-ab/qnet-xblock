• Defines title in the horizontal navigation at the top of the page
  • Defines question or subtitle for the lab
  • Defines description for the lab
  • Defines maximum number of points student can get
  • Defines used Qnet platform domain (https://qnet.bonch-ikt.ru/)
  • Defines used path to the Qnet platform lab,
    relative to domain and should has an element parameter (stand/{element})
  • Defines used Qnet platform element for the lab
  • Defines used Qnet platform api method for getting user progress, relative to domain and
    should has a user_name, element and token parameters
    (api/progress?user_login={user_name}&element={element}&token={token})
  • Defines used Qnet platform api method for getting lab status, relative to domain and
    should has a user_name, element and token parameters
    (api/wstatus?user_login={user_name}&element={element}&token={token})
  • Defines used iteration id for requests to the Qnet platform,
    by default is similar with used lab element
